# MFevziDemir Solutions - Customer Portal & Portfolio

Bu proje, **MFevziDemir Solutions** müşterileri için geliştirilmiş, modern, güvenli ve gerçek zamanlı bir müşteri yönetim portalıdır. Aynı zamanda kurumsal portföy ve hizmet tanıtım sitesini de içerir.

## 🚀 Öne Çıkan Özellikler

- **Modern Glassmorphism Tasarım:** Tamamen responsive ve göz yormayan, premium cam efekti arayüzü.
- **Firebase Entegrasyonu:** Gerçek zamanlı Firestore veritabanı ve güvenli Authentication sistemi.
- **Gelişmiş Müşteri Paneli:**
  - **Canlı Destek Sistemi:** Kurucu (Founder) ile müşteriler arasında anlık mesajlaşma.
  - **2FA (İki Faktörlü Doğrulama):** TOTP tabanlı üst düzey hesap güvenliği.
  - **Hesap Yönetimi:** Güçlü şifre ölçerli şifre yenileme ve güvenli e-posta güncelleme (Verification-based).
  - **E-posta Doğrulama:** Sahte hesapları önleyen mail doğrulama sistemi.
- **Founder Dashboard:** Adminler için gelen talepleri yönetme, kullanıcılara yanıt verme ve sunucu istatistiklerini (Verinox API) takip etme paneli.
- **SEO & Performans:** Modern SEO teknikleri ve yüksek sayfa yükleme hızı optimizasyonu.

## 🛠️ Kullanılan Teknolojiler

- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+), Bootstrap 4.6.
- **Backend/DB:** Firebase Auth, Firestore.
- **Güvenlik:** OTPLib (2FA), Firebase Security Rules.
